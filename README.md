# OIB-SIP
🍥 Naruto Shippuden Website

A simple Naruto Shippuden themed website built using HTML and CSS. It showcases the anime’s story, characters, and powers with a clean dark design.

🚀 Features
Dark theme UI
Navigation bar with section links
Character cards (Naruto, Sasuke, Sakura)
Powers and abilities section
Clean layout using Flexbox
🛠️ Tech Used
HTML
CSS
